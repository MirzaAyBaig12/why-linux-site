What I Use: Pop!_OS 24.04 LTS

I use Pop!_OS 24.04 LTS, which is a Linux distribution based on Ubuntu. It is made by System76 and is designed to be stable, fast, and easy to use while still giving you full control over your system.

One reason I like it is because it feels clean and organized. It doesn’t come with a lot of unnecessary apps or clutter, but it still has everything you need for daily use like browsing, coding, or general work. It also runs smoothly and feels reliable. Another thing I noticed is that it manages system resources well. Even when I have multiple things open, it doesn’t feel heavy, and it doesn’t make my fans sound like a tractor.

Overall, Pop!_OS works well for me because it is simple when I need it to be, but still powerful when I want to customize things. It gives a good balance between ease of use and control.

COSMIC Desktop Environment

COSMIC is the desktop environment used in Pop!_OS, developed by System76. It is designed to be a modern replacement for traditional desktop environments and focuses on making the system feel faster, cleaner, and more efficient to use. It is currently in beta, with the first stable release, COSMIC Epoch 2, aimed for a window of July - August.

A desktop environment is basically the interface you use to interact with your system. It includes things like the file manager, taskbar, windows, and menus. On Windows this would be File Explorer, and on macOS it would be Finder and the desktop interface. The key difference is that on Linux, the desktop environment is not a fixed part of the system.

Instead, you can choose or swap desktop environments depending on what you want. This means you are not forced into one interface style. You can install environments like GNOME, KDE, or XFCE, and completely change how your system looks and behaves without changing the core system itself.

I also customized COSMIC to better fit my workflow. I adjusted it to resemble a macOS-style layout with a top panel and dock-like setup. This shows how flexible Linux desktops can be, since you can change not only performance and tools, but also the entire visual layout of your system.

Overall, COSMIC is designed to feel simple and fast while still giving you control over your workflow, making it a good fit for a system that values flexibility.
